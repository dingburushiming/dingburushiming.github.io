原生的js所获取的对象是DOM对象，jQuery所获取的对象是一个伪数组jQuery对象,两类对象都不可以使用对方的属性，方法。

js对象和jQuery对象之间是可以相互转换的,转化后可以使用对方的函数属性

js-> jQuery(用$符号包裹)

var video = document.queryselector(‘video’);

$(video); //变量不能用分号括起来

var divsss = document.querySelector(‘div’);

​ $(divsss).hide();

jQuery->js

$(‘video’)[0];

$(‘video’).get(0);

$(‘div’).get(0).style.display = ‘none’;

      jQuery中的拷贝：

      $.extend([deep], target, object1)

      1.deep默认为false浅拷贝，参数为true则是深拷贝

      2.target时拷贝到的对象，object1时被拷贝对象

      3.浅拷贝只是将被拷贝对象的复杂数据类型（对象）中的地址给拷贝对象，更改数据对两个对象都有影响

      var data = {

      ​ name: “张三”,

      ​ pasword: “123”,

      ​ msg: {

      ​ data: “shf”

      ​ }

      };

      var target = {};

      $.extend(true,target,data);

      target.msg.data = “李四”;//只有复杂的对象的浅拷贝才会通过地址

      4.深拷贝是将被拷贝对象的数据完全克隆，而不是拷贝地址，更改数据只会影响更改的对象
